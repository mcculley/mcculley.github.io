Apple Keyboard Shortcuts

Most of these shortcuts will work with all standard applications on macOS and iOS.

Typesetting

^ control ⌘ + superscript (e.g., x2)
^ control ⌘ - subscript (e.g., H2O)

Miscellaneous Symbols

⌥ option 4 ¢ (cent)
⌥ option ⇧ shift 8 ° (degree)
alt 2 ™ (trademark)
⌥ option ⇧ shift k  (apple)

Screenshots

add ^ control to save to the clipboard instead of the desktop

⌘ ⇧ shift 3 a screenshot of the entire screen is added to your desktop
⌘ ⇧ shift 4 a screenshot of an area is added to your desktop
⌘ ⇧ shift 4 space a screenshot of a specified window is added to your desktop

Text Navigation & Selection

add ⇧ shift to select text between cursor and end point

⌘ ← go to beginning of line
⌘ → go to end of line
⌥ option ← go to beginning of current word
⌥ option → go to end of current word
⌥ option ↑ go to beginning of document
⌥ option ↓ go to end of document
