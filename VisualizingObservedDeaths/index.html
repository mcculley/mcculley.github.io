This is a visualization, using polar coordinates, of the total observed deaths for every week using data published by CDC.

Using this kind of graph, one can more easily see how changes year-to-year in total observed deaths have been affected by COVID-19. Using total observed deaths, one can disregard the (mis)attribution of deaths to COVID-19.

Source code for the tool that generates these graphs is available at https://github.com/mcculley/ObservedDeathVisualizer.
